# agregacion
¿Que es? es un tipo especial de asociación que representa una relación "todo-parte", donde la parte puede existir independientemente del todo.
¿Que hace?  representa una asociación "todo-parte" o "tiene un", donde un objeto (el contenedor o "todo") agrupa a otros (las partes), pero estos componentes son independientes y pueden existir sin el contenedor
¿Como lo hace?  Se representa con una línea que tiene un rombo vacío en el extremo de la clase contenedora.
Requerimentos técnicos: La relación de agregación en el modelado técnico (UML) representa una forma especializada de asociación que describe una relación "todo-parte", donde un objeto contenedor (el "todo") está compuesto por otros objetos componentes (las "partes")
